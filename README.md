# Personal portfolio

This portfolio was built as a part of my Bachelor's Thesis. The aim of the project was to display my existing skills with web development and to learn new. During this project I learned a lot, especially about testing and CI/CD automation.

The project is built by using TypeScript, React and Vite. Component and integration testing is done using Jest and RTL, and E2E-testing is done using Cypress. Learning testing tools was one of the main motivations for this project so I wanted to use multiple. 

The portfolio is deployed on GitHub pages (mkytomies.github.io/Portfolio), deployment happens on every push on main branch using GitHub Workflows. 

UI design was done in Figma (https://www.figma.com/community/file/1623305769059810923). The goal of the UI design was to be simple and to make the home page resemble my CV. In the future I hope to make a new design for this that will be more fun and maybe more challenging to implement. Some fun stuff with animations etc.
